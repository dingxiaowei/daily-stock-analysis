# 六年上万业余时间量化研究坚持的心路历程
```
“你是否曾有一个梦想，在别人眼中遥不可及，甚至被质疑、被否定，但你内心深处却坚信，只要坚持就能实现？我就有这样一段经历。

六年前，我萌生了开发一款智能量化软件的想法。那时候，19年金融市场风起云涌，我看到了人工智能量化交易背后的巨大潜力，deepseek ceo梁文峰就是同一时间量化成功的优秀典范。于是，在繁忙的工作之余，我开启了这条充满挑战的研究之路。白天，我是公司里那个为了业绩奔波的职员；夜晚，当整个城市都陷入沉睡，我却坐在电脑前，对着一行行代码，夜以继日地钻研。上万小时的业余时间，我几乎废寝忘食，无数个夜晚，窗外的星星是我唯一的陪伴。

一开始，朋友们并不理解我的做法。他们觉得我的想法太宏大不切实际，非要在业余时间折腾这么个软件，简直是异想天开。有人劝我早点放弃，说这根本不可能成功，还有人质疑我是不是脑子糊涂了。面对这些质疑，我没有过多地去解释。因为我知道，语言是苍白的，只有用行动和结果才能证明一切。

在开发的过程中，我遇到了无数的难题。技术上的瓶颈、精力远远不够（曾经我连续一周每天只吃一顿，只睡两小时，疯狂的迭代开发），每一个问题都像一座大山，压得我喘不过气来。但我从未想过放弃，每一次遇到困难，我都会告诉自己：再坚持一下，再努力一点。就这样，我一步一个脚印，克服了一个又一个难关。

终于，在经历了六年的漫长时光后，我的智能量化软件终于开发成功了！当我把软件展示给朋友们看时，他们的态度发生了180度的大转变。一开始的质疑变成了惊叹，那些曾经劝我放弃的人，现在都对我竖起了大拇指，纷纷说真香。

通过这段经历，我深刻地明白了一个道理：不要试图去说服别人，因为每个人都有自己的认知和判断。当你有了一个目标，就勇敢地去行动，用结果去征服他们。在这个世界上，没有什么比实际的成果更有说服力。

朋友们，如果你也有一个梦想，不要害怕别人的质疑，不要被外界的声音所干扰。拿出你的勇气和毅力，像我一样，夜以继日地去努力，去奋斗。我相信，只要你坚持下去，用行动和结果说话，你一定能够实现自己的梦想，让那些质疑你的人刮目相看！现在，就行动起来吧！”
```
